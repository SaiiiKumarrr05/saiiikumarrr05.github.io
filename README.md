# Saikumar Potturi — Portfolio Website 🚀

A personal portfolio website built with pure HTML, CSS, and Three.js — no frameworks, no build step, no npm. Just drop and deploy.

**Live Site:** (https://saiiikumarrr05.github.io/)

---

## Preview

> Dark, high-end aesthetic with a real-time 3D background — floating wireframe polyhedra, orbital rings, star field, and mouse parallax powered by Three.js.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, animations, grid) |
| 3D Background | Three.js (r128) |
| Fonts | Syne + Space Mono (Google Fonts) |
| Hosting | GitHub Pages |

---

## Features

- 🌐 **Three.js 3D background** — 20 floating wireframe polyhedra, 3 orbital rings, 400-point star field
- 🖱️ **Custom cursor** with magnetic hover effect
- 📜 **Scroll-triggered reveal animations**
- 🎞️ **Infinite marquee** skills strip
- 📱 **Fully responsive** (mobile-friendly)
- ⚡ **Zero dependencies** — no npm, no build tools

## Getting Started

```bash
git clone https://saiiikumarrr05.github.io/
# open index.html in your browser — no server needed
```

---

## Deployment (GitHub Pages)

1. Push `index.html` to the `main` branch
2. Go to **Settings → Pages → Source → main / root**
3. Save — your site goes live at:

```
https://saiiikumarrr05.github.io/
```
